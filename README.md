# Hospital Reservation 웹사이트

대한민국 1위 비대면 진료 앱 - Hospital Reservation 랜딩 페이지

## 프로젝트 구조

```
hospital-reservation/
├── index.html          # 메인 HTML 파일
├── css/
│   ├── style.css       # 메인 스타일시트
│   └── responsive.css  # 반응형 스타일시트
├── js/
│   └── main.js         # 메인 JavaScript 파일
└── README.md           # 프로젝트 설명
```

## 기능

- 반응형 디자인 (데스크탑, 태블릿, 모바일)
- 모바일 햄버거 메뉴
- 스크롤 애니메이션 (Intersection Observer)
- 숫자 카운터 애니메이션
- 후기 슬라이더 (터치 스와이프 지원)
- 부드러운 스크롤 네비게이션
- 서비스 카드 순차 등장 애니메이션

## 사용법

VS Code에서 폴더를 열고 `index.html`을 Live Server로 실행하세요.

### VS Code에서 실행하기
1. VS Code에서 폴더 열기 (File > Open Folder)
2. Live Server 확장 프로그램 설치
3. `index.html` 우클릭 > "Open with Live Server"
